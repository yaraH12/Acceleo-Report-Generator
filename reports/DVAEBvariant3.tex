	\section{Introduction}
	This assurance-case report presents the results of the 
	representativity evaluation performed on the different model variants considered for 
	the simulation need. The document is structured to provide a clear and traceable view of how 
	each model was assessed. For every model variant, the report lists all associated requirements
	 and specifies whether each requirement has been successfully satisfied. Based on these 
	individual outcomes, the report then summarizes the performance of each model using the 
	representativity factors defined in our framework, expressed both as percentage scores 
	and corresponding levels on the assessment scale. These elements allow the reader to understand 
	the strengths and limitations of each variant.
