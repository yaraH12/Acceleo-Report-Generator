This repository contains an Acceleo‑based code generator designed to produce structured reports from .xassure verification result files. The generator parses the content of an XAssure object, extracts relevant verification information, and automatically generates human‑readable reports.
The goal of this project is to streamline the analysis of verification results produced by OSATE/Assure and provide a consistent reporting format that can be integrated into documentation workflows or model‑based engineering pipelines.

Parses .xassure files generated by OSATE’s Assure engine. Analyzes XAssure objects, including:
verification outcomes - passed/failed checks - associated claims and properties - contextual metadata - Generates structured reports using Acceleo templates - Supports customizable report formats (extend or modify templates as needed)
Fully model‑driven: relies on EMF models and Acceleo M2T transformations.

How it works:
Provide an .xassure file : The generator takes as input a verification result file produced by OSATE/Assure.
Acceleo loads the XAssure model : The .xassure file is parsed as an EMF model instance.
The generator analyzes the object. It extracts: verification results - evidence - claims - messages - metadata
